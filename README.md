# 礁溪遊記小網站

這是可部署到 Vercel / Netlify / GitHub Pages 的 React + Vite 專案。

## 本機預覽
```bash
npm install
npm run dev
```

## 建置
```bash
npm run build
```

## 上線建議
最簡單方式：上傳到 GitHub 後，用 Vercel 匯入專案。
Build Command：`npm run build`
Output Directory：`dist`

## 功能
- 每日行程
- 備用景點
- Google Maps 從目前位置導航
- 預設路線導航
- 手機版介面
